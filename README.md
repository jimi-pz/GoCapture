# GoCapture

A .NET console application for testing the capture of a piece in GO board game.

## Project Overview

It provides a structured way to think about the game and the capture algorithm.

## Getting Started

### Prerequisites

* [.NET 10.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0) (Updated from .NET 5.0)
* JetBrains Rider or Visual Studio

### Building the Project

To build the project, run the following command in the root directory:

```bash
dotnet build
```
